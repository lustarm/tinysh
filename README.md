# TinySh - A Simple C-based Shell

TinySh is a basic shell program written in C.

## Features

- Basic command execution
- Input/output redirection
- Background process support
- Built-in commands
- Signal handling (Ctrl+C)
- Command history
- Tab completion

## Usage

1. Clone the repository.
2. Compile the code.
3. Run `./tinysh`.
